# RTL Source Set

This directory contains the synthesizable Verilog source files for
`tt_um_shivamtiwari020505_i2c`.

Tiny Tapeout source inclusion is controlled by `../info.yaml`. The RTL top
module is `tt_um_shivamtiwari020505_i2c.v`, which wraps the internal APB I2C
master/slave core with the byte-wide Tiny Tapeout host interface.
